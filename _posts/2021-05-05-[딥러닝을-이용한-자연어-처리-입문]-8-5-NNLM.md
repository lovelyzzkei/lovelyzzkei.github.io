우리가 새로운 언어를 배울때 학원을 다니기도 하고, 유튜브에 있는 좋은 영상들을 찾아보기도 하지만, 모두가 인정하는 최고의 방법은 그 언어를 사용하는 나라에 가서 1년 정도를 사는 것입니다. 직접 그 나라 사람들이랑 얘기를 하며 그 언어를 습득하는 것이 가장 빠르면서 효과적입니다.  

기계도 마찬가지입니다. 우리는 우리가 사용하는 언어의 문법이나 의미들을 그리 잘 지키지 않으며 사용합니다. 그런 기계에게 정석적으로 문법이나 단어들을 가르치는 것보다 우리가 일상에서 사용하는 대화들을 가지고 학습시키는 것이 더 효율적입니다. 그래서 최근에는 과거의 통계적인 접근보다 **인공 신경망**을 사용하는 모델이 더 각광받고 있고, 더 좋은 성능을 내고 있습니다.   

이번 시간에는 인공 신경망을 사용한 시초 모델인 **피드 포워드 신경망 언어 모델(Feed Forward Neural Network Language Model, NNLM)**에 대하여 공부해보겠습니다.  

## NNLM의 등장 배경

우리가 앞에서 배운 언어 모델에는 통계적 접근 기반의 **N-gram 언어 모델**이 있습니다. 하지만 통계적 접근 기반의 언어 모델은 충분한 데이터를 관측하지 못하면 언어를 제대로 모델링하지 못하는 **희소 문제(Sparsity problem)**를 가지고 있었습니다. 훈련 데이터에 없는 단어 시퀀스에 대한 예측을 할 수 없었던 것이죠. 이는 N-gram 언어 모델도 마찬가지였습니다. 또, 빈도 수 기준으로 모델링되기 때문에 **단어 간 유사성**을 학습할 수도 없었습니다.  

만약, 언어 모델이 단어의 유사도를 학습할 수 있도록 설계된다면, 훈련 데이터에 없는 단어 시퀀스에 대한 예측이라도 유사한 단어가 사용된 단어 시퀀스를 참고하여 보다 정확한 예측이 가능해질 것입니다. 예를 들어보죠. '송독하다'라는 단어는 '소리를 내어 글을 읽다'라는 생소한 단어입니다. 모델이 '읽다'와 '송독하다'가 유사한 의미를 갖고 있는 것을 학습한다면, '책을 송독하다' 라는 데이터를 훈련하지 않았더라도 '책을 읽다'와 유사한 의미를 갖고 있다고 판단하여 예측할 수 있을 것입니다. 이런 아이디어를 가지고 탄생한 모델이 바로 **NNLM**입니다.  

## NNLM

책에서 설명한 예시를 가지고 NNLM의 학습 과정을 살펴보며 NNLM에 대하여 알아보겠습니다.  

<img src="https://github.com/lovelyzzkei/lovelyzzkei.github.io/blob/master/assets/img/8_5_nnlm.jpg?raw=true">

언어 모델은 주어진 단어 시퀀스로부터 다음 단어를 예측하는 모델입니다. 위의 예문 'what will the fat cat sit on'에서 'what will the fat cat'이 주어졌을때 'sit'을 예측하는 과정을 알아봅시다.  

일단, 기계는 문자보다는 숫자로 된 표현을 더 잘 처리하기 때문에 먼저 예문을 수치화하겠습니다. 이 과정에는 앞에서 배웠던 **원-핫 인코딩**을 사용합니다.  

```python
what = [1, 0, 0, 0, 0, 0, 0]
will = [0, 1, 0, 0, 0, 0, 0]
the = [0, 0, 1, 0, 0, 0, 0]
fat = [0, 0, 0, 1, 0, 0, 0]
cat = [0, 0, 0, 0, 1, 0, 0]
sit = [0, 0, 0, 0, 0, 1, 0]
on = [0, 0, 0, 0, 0, 0, 1]
```

NNLM은 N-gram과 유사하게 다음 단어를 예측할 때, **앞의 n개의 단어만 참고**합니다. 이때 n을 **윈도우(window)**라고도 합니다. 이번 예시에서는 n을 4라고 가정하겠습니다. 그러면 NNLM은 `sit`을 예측하기 위해서 앞에 4개의 단어, `will`, `the`, `fat`, `cat`을 참고합니다.   

4개의 원-핫 벡터는 입력층을 지나 **투사층(Projection layer)**으로 들어옵니다. 투사층은 입력값에 가중치 행렬을 곱하지만 활성화 함수는 존재하지 않습니다. 단어 집합의 크기를 $V$라고 하고, 투사층의 크기를 $M$이라고 하면, 행렬 곱의 특성상 가중치 행렬을 $V x M$의 크기를 가집니다. 위의 예시에서는 단어가 7개 이므로 $V=7$이고, $M$은 5라고 가정하면 가중치 행렬 $W$는 $7x5$의 크기를 가집니다. 그리고 입력 단어의 벡터가 원-핫 벡터이기 때문에 가중치 행렬과 곱하면, 입력 단어 벡터에서 1을 가지는 인덱스에 해당하는 가중치 행렬의 행이 그대로 반환됩니다. 이를 **룩업 테이블(lookup table)**이라고도 합니다.  

위 과정을 통해 크기가 $M$으로 압축된 각각의 단어 벡터들은 투사층 내에서 **연결(concatenate)**되어 우리가 알고 있는 은닉층으로 향합니다. 은닉층 내의 활성화 함수를 거치고 난 후 출력층의 활성화 함수를 거쳐 최종적으로 모델의 예측값이 등장합니다. 이 값을 실제값에 가깝게 하기 위하여 손실 함수를 가지고 가중치 업데이트가 이루어집니다. 이때 업데이트되는 가중치들이 바로 **투사층에 있는 가중치 행렬**입니다. 가중치가 계속 업데이트 되면서 투사층을 지나는 단어 벡터들도 계속 바뀌게되는데 이 단어 벡터들을 **임베딩 벡터(Embedding vector)**라고도 부릅니다. 이 과정을 반복하면서 학습해나가는 것이 NNLM의 학습 과정입니다.  

NNLM은 충분한 양의 데이터를 학습할 경우 **유사한 목적으로 사용되는 단어들이 유사한 임베딩 벡터값을 얻기 때문에** 예측 과정에서 훈련 코퍼스에 없는 단어 시퀀스라도 예측이 가능해집니다. 그리고 기존에 N-gram이 가지고 있었던 **단어의 유사성** 및 **희소 문제**를 해결할 수 있습니다.  

하지만 N-gram과 유사하게 여전히 **이전의 n개의 단어만 참고**하기 때문에 버려지는 단어들이 가지는 **문맥 정보**를 참고할 수가 없다는 한계점이 있습니다.
